# 小红书工具箱

小红书生态工具集合，包含博主蒸馏、博主分析、帖子深挖、文案改写等工具，统一 Web 界面操作。

## 功能

### 🔧 工具类

| 工具 | 说明 |
|------|------|
| **博主蒸馏** | 采集博主笔记 → 数据分析 → 深度蒸馏报告。需 TikHub Token |
| **博主分析** | 使用本地爬虫分析博主风格，无需 Token。需配置 Cookie |
| **帖子深挖** | 分析单条帖子的评论区情感和观点 |
| **文案改写** | 小红书文案风格指南，供参考仿写 |

### 📚 写作库

已分析的博主竞品材料，包括：
- 粗门、粗趣、薯岛、取伙
- 顽聚桌游社、深圳华侨城创意文化园

每份分析包含：笔记数据、分析报告、创作指南

## 快速开始

### Windows

双击 `setup.bat`，然后：

```bash
python app.py
```

浏览器打开 http://localhost:5000

### macOS / Linux

```bash
pip install -r requirements.txt
python app.py
```

## 前置依赖

| 依赖 | 用途 | 获取方式 |
|------|------|---------|
| Python 3.8+ | 运行环境 | python.org |
| Node.js | 爬虫 JS 签名 | nodejs.org |
| 小红书 Cookie | 本地爬虫 | 浏览器登录后复制 |
| TikHub Token | 云端采集 | tikhub.io（可选） |

## 项目结构
