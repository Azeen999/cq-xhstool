🔑 Cookie 管理

×

小红书的 Cookie 会过期，过期后需要重新获取并粘贴到这里。

📖 如何获取 Cookie？
1. 浏览器登录 小红书
2. 按 F12 打开开发者工具 → Network（网络）标签
3. 刷新页面，点击任意一个请求（如 feed 或 homefeed）
4. 在 Request Headers 中找到 Cookie: 那一行
5. 右键 → Copy value → 粘贴到下方输入框
6. 点击保存
