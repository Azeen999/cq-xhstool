# 小红书博主风格分析器

> ⚠️ **使用前必读**：本工具使用本地爬虫获取公开数据，需提前配置 Cookie。仅供学习研究使用。

## ⛔ 执行前铁律

触发分析任务前，以下两项必须由用户**明确说出**：
- 平台：小红书（目前仅支持）
- 输入：博主主页链接 **或** 博主ID

---

## 你是什么

基于本地爬虫的小红书博主风格分析工具。**输入一个博主主页链接或博主ID，输出三样最终产物：**

1. **博主笔记文件夹** — 采集到的笔记正文和图片列表
2. **HTML 分析报告** — 给人看，快速理解博主的风格特点
3. **创作 Skill 文件夹** — 给 AI 用，安装后可模仿该博主风格创作

---

## 能力范围

### 分析维度

| 维度 | 内容 |
|------|------|
| **人设画像** | 博主定位、语言风格、视觉风格 |
| **内容策略** | 选题方向、发布节奏、互动策略 |
| **标题公式** | 常用标题模式、关键词偏好 |
| **正文结构** | 开头模板、内容框架、CTA 方式 |
| **标签策略** | 高频标签、话题偏好 |

### 产出物

1. **笔记数据**：`{博主名}_笔记/` 文件夹，包含每篇笔记的正文和图片列表
2. **分析报告**：`{博主名}_分析报告.html`，可视化展示博主风格
3. **创作指南**：`{博主名}_创作指南.skill/SKILL.md`，供 AI 使用的创作模板

---

## 前置要求

- Python 3.10+
- 小红书 Cookie（需手动配置到 `.env` 文件）
- Node.js（爬虫签名依赖）
- 网络连接

### Cookie 配置

在 `D:\myproj\tools\Spider_XHS\.env` 文件中配置：
```
XHS_COOKIE=你的小红书Cookie
```

---

## 执行流程

### Phase 1: 环境检查
- 检查 Python 版本
- 验证 Cookie 是否配置

### Phase 2: 数据采集
- 输入：博主主页链接 或 博主ID
- 采集目标：最多 50 篇笔记（按点赞数排序）
- 保存路径：`./output/{博主名}_笔记/`

### Phase 3: 风格分析
- 标题模式识别
- 正文结构分析
- 标签统计
- 互动数据分析

### Phase 4: 产出物生成
1. HTML 分析报告
2. 创作 Skill 文件夹

## 使用方式

### 自然语言触发（推荐）

直接对 AI 说：
```text
分析博主 https://www.xiaohongshu.com/user/profile/xxx
```
或
```text
蒸馏博主 54701b42d6e4a942975b8059
```

### 命令行运行

```bash
cd skills/xhs-blogger-analyzer/
python run.py --url "https://www.xiaohongshu.com/user/profile/xxx"
```

### 参数说明

| 参数 | 说明 | 必填 |
|------|------|------|
| --url | 博主主页链接 | 二选一 |
| --name | 博主名（会自动搜索） | 二选一 |
| --max-notes | 最大采集数量（默认 50） | 否 |
| --output | 输出目录（默认 ./output） | 否 |

---

## 注意事项

1. **Cookie 有效期**：Cookie 可能会过期，需要定期更新
2. **采集频率**：内置随机延迟（3-8秒/篇测试，生产建议15-30秒），避免被风控
3. **数据准确性**：基于公开数据采集，可能存在一定误差
4. **仅供学习**：本工具仅供学习研究使用，请遵守平台规则
