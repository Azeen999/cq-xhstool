# 小红书帖子深度分析器

一键分析小红书帖子的评论区，提取用户观点和情感倾向。

## 使用场景

- **舆情监控**：追踪特定帖子的用户反馈
- **竞品分析**：了解竞争对手帖子的评论反应
- **内容优化**：从评论中获取改进建议
- **用户洞察**：了解目标用户的真实需求

## 工作流程

### Step 1: 接收输入

用户提供：
- **帖子链接**：小红书帖子URL（格式：`https://www.xiaohongshu.com/item/xxx`）

### Step 2: 数据采集

自动获取：
- 帖子标题、正文、作者信息
- 互动数据（点赞、收藏、评论数）
- 评论列表（支持分页获取）

### Step 3: 智能过滤

自动过滤无意义评论：
- 点赞数 < 3 的评论
- 内容长度 < 10 字的评论
- 纯表情、无意义内容

### Step 4: 分析处理

- **情感分析**：正面/中性/负面/提问分类
- **关键词提取**：提取评论热词
- **观点分类**：正面观点、负面观点、用户提问、改进建议

### Step 5: 生成报告

输出结构化Markdown报告，包含：
- 帖子基本信息
- 完整正文内容
- 评论分析概览
- 热门评论TOP10
- 评论热词表
- 用户观点汇总

## 技术依赖

- **数据采集**：Spider_XHS工具
- **Cookie配置**：`D:\myproj\tools\Spider_XHS\.env`
- **输出目录**：`D:\myproj\参考\`

## 使用示例

```
用户：帮我分析这个帖子 https://www.xiaohongshu.com/item/123456789

→ 自动采集帖子信息和评论
→ 智能过滤无效评论
→ 情感分析和关键词提取
→ 生成 Markdown 报告
```

## 注意事项

1. **Cookie有效性**：确保小红书Cookie未过期
2. **评论数量**：受API限制，最多获取约200条评论
3. **分析准确性**：情感分析基于关键词匹配，可能存在误差
