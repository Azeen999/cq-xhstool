# 素材库

此目录用于存储博主分析结果和创作素材。

## 目录结构

```
素材库/
├── 自我参数/           # 自己账号的分析结果（不上传）
│   ├── 创作指南.skill/  # 博主风格蒸馏结果
│   ├── 分析报告/        # 蒸馏报告 HTML
│   ├── 笔记/           # 采集的笔记内容
│   └── *.json          # 分析数据
└── ...
```

## 使用说明

1. 运行博主蒸馏后，结果会保存在 `自我参数/{博主名}/` 目录下
2. 创作指南 `SKILL.md` 可用于文案改写时参考风格
3. 此目录下的敏感数据不会上传到 GitHub

## 添加新博主

在 Web 界面的「博主分析」标签页输入博主主页链接即可。
