# 粗趣活动文案助手

> 生成粗趣活动招募帖，基于活动模板库 + 品牌风格指南

## 知识库文件

技能运行时读取以下知识库文件获取模板和参考：

- `素材库/自我参数/粗趣/活动模板库.md` — 按活动类型的标题/正文/标签模板
- `素材库/自我参数/粗趣/活动运营SOP.md` — 全流程标准化手册
- `素材库/自我参数/粗趣/常见活动Q&A.md` — 用户常见问答
- `素材库/自我参数/粗趣/创作指南.skill/SKILL.md` — 品牌风格指南
- `素材库/自我参数/粗趣/粗趣答疑帖.md` — 已有问答素材

## 触发

「写活动招募」「生成活动文案」「帮我写活动帖」「粗趣活动」

## 工作流程

### Step 1：收集活动信息

向用户收集以下信息（如果用户已提供则跳过）：

```
活动类型：【桌游局 / 主题派对 / 盲盒社交 / 户外活动 / 工作坊手工】
活动时间：（日期+时间）
活动地点：（区域+具体地址）
活动费用：（价格）
人数上限：（数字）
活动亮点：（选填，1-2句话）
目标人群：（默认"深圳年轻人"）
补充说明：（选填）
```

如果用户只给了部分信息，主动追问缺失的字段。

### Step 2：读取知识库模板

读取 `活动模板库.md`，找到对应活动类型的模板章节，包括：
- 标题模板（3-5个公式）
- 正文结构模板
- 标签推荐
- 互动话术

同时读取 `创作指南.skill/SKILL.md` 中的品牌风格要求。

### Step 3：生成活动文案

基于用户信息和模板，生成完整的小红书活动招募帖。输出格式：

```
**标题**（不超过20字）

【正文】
（口语化、短句分行、emoji点缀、200字左右）

🔥 活动亮点
- ...

📍 地点：...
⏰ 时间：...
🎲 内容：...
💰 费用：...
👥 人数：...
👇 报名方式：点下方链接报名

（结尾互动）

【标签】
#深圳 #活动类型 #社交 #周末去哪儿 #粗趣
```

### Step 4：输出 + 可编辑

展示生成结果后，主动询问：
- 「需要调整标题吗？」
- 「要加/删某个环节吗？」
- 「要换一种风格吗？」
- 「要生成不同版本的标题吗？」

## 品牌规范

### 违禁词（绝对不能出现）
❌ 第一/最佳/唯一/顶级/极致/绝对
❌ 私信/V/微信/卫星/扣1/暗号
❌ 包教包会/保证/100%有效
❌ 史上/全网最低价

### 合规替换
私信/V/微信 → 点下方链接报名
包教包会 → 专人带教，带你上手

### 风格要求
- 口语化、短句分行
- 适当使用 emoji 点缀
- 有安利感、情绪共鸣
- 结尾引导互动+报名
